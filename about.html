Over Ons Project

Dit project is ontwikkeld als een eenvoudig weerdata-dashboard waarmee gebruikers temperatuur, vochtigheid en het weerstype kunnen invoeren. De data wordt opgeslagen en kan op een overzichtelijke manier worden weergegeven.

Het doel van dit project is om te leren hoe je interactieve webapplicaties maakt met HTML, CSS, JavaScript en PHP, inclusief basisgebruik van databases en sessiebeheer.
